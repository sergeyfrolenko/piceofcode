Ошибки при верстке

  1. соблюдайте семантику
  2. логичное использование margin/padding
  3. не указывайте высоту для текстовых блоков
  4. контроль line-height:1.2; для элементов с большим шрифтом
  5. адаптировать картинки width and height
  6. font-family: inherit; font-size: inherit; для наследования от body
  7. textarea resize:none;
  8. input :focus стилизация
  9. стилизация в css с помощью псевдоэлементов
